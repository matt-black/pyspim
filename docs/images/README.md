## Image Guidelines

### Screenshot Requirements
- **Resolution**: Minimum 1200x800 pixels, preferably 1920x1080 or higher
- **Format**: PNG format for best quality
- **File Size**: Keep under 2MB per image for web loading
- **Quality**: High quality, clear screenshots with readable text

### Content Guidelines
- **Show Complete Interface**: Include relevant UI elements and controls
- **Include Context**: Show the plugin in context with napari viewer when relevant
- **Consistent Styling**: Use consistent window sizes and layouts across images

## Upload Instructions

1. Take screenshots of the plugin interface
2. Save them with the exact filenames listed above
3. Place them in this `docs/images/` directory
4. Ensure images are optimized for web (PNG format, reasonable file size)
